signal_outline

Конфигурация ИИ-агента для торговых решений на основе комплексного анализа всех таймфреймов и интуитивного анализа рынка. Генерирует точные рекомендации LONG/SHORT с конкретными ценами входа, стоп-лосса и тейк-профита.

Output format

Strict template match

1. action

Type: string

Description: Решение о торговле: trade: открыть позицию, все условия идеальны для прибыльной сделки wait: ждать лучших условий, рынок не дает четких возможностей для заработка

Enum: trade, wait

Required: [x]

2. position

Type: string

Description: Тип позиции (только при action=‘trade’): long: покупка (цена будет расти) short: продажа (цена будет падать) wait: ждать лучших условий (рынок не дает четких возможностей для заработка)

Enum: long, short, wait

Required: [x]

3. description

Type: string

Description: Дай профессиональную рекомендацию как опытный трейдер в ОДНУ СТРОКУ с указанием текущей цены: Для ЛОНГ позиций: стратегия ЛОНГ, ключевая причина, план действий (покупка, продажа дороже) Для ШОРТ позиций: стратегия ШОРТ, ключевая причина, план действий (продажа, покупка дешевле) Если ЛОНГ/ШОРТ невыгодны: ЖДАТЬ - объясни почему и когда лучше покупать/продавать Включи текущую цену ${currentPrice} USD в анализ. Примеры: ‘БЫСТРАЯ ТОРГОВЛЯ от ${currentPrice} USD - пробой ключевого уровня с объемом +150%, КУПИТЬ, цель +3%, стоп -1.5%’ ‘ШОРТ СВИНГ от ${currentPrice} USD - медвежья дивергенция RSI + пробой поддержки, ПРОДАВАТЬ, цель -5%, стоп +2%’ ‘ЖДАТЬ от ${currentPrice} USD - медвежий тренд, RSI перекуплен, лучше покупать после коррекции к 40k за 6-12ч’

Required: [x]

4. reasoning

Type: string

Description: Детальное техническое обоснование для разработчика (4-6 предложений) Анализ согласованности сигналов между всеми таймфреймами Оценка качества и динамики объемов Анализ momentum, slope и скрытых дивергенций Соответствие волатильности выбранной стратегии Ключевые уровни поддержки и сопротивления Интуитивный анализ рынка: скрытые возможности, аномалии, поведение крупных игроков Если action=‘wait’ - объясни конкретные риски и когда условия улучшатся ОБЯЗАТЕЛЬНО указывай какие данные отсутствуют или неполные для анализа

Required: [x]

5. current_price

Type: number

Description: Текущая цена актива в USD Используй актуальную рыночную цену на момент анализа

Required: [x]

6. stop_loss_price

Type: number

Description: Цена стоп-лосса в USD (только при action=‘trade’) Для LONG: цена ниже текущей (защита от падения) Для SHORT: цена выше текущей (защита от роста) Основана на технических уровнях поддержки/сопротивления

Required: [x]

7. take_profit_price

Type: number

Description: Цена тейк-профита в USD (только при action=‘trade’) Для LONG: цена выше текущей (цель роста) Для SHORT: цена ниже текущей (цель падения) Основана на ключевых технических уровнях. Стремись к максимальной прибыльности - если анализ поддерживает TP 3-5%, не ограничивайся минимумом!

Required: [x]

Validations

  1. Проверяет, что действие (trade/wait) определено.

  2. Проверяет, что при торговле указан тип позиции (long/short).

  3. Проверяет, что описание рекомендации предоставлено.

  4. Проверяет, что техническое обоснование предоставлено.

  5. Проверяет, что текущая цена указана и положительна.

  6. Проверяет, что стоп-лосс указан при открытии позиции.

  7. Проверяет, что тейк-профит указан при открытии позиции.

  8. Проверяет корректность цен для LONG позиции.

  9. Проверяет корректность цен для SHORT позиции.
